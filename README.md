# Level Up

Level Up is an online learning platform that offers a wide range of courses to help you reach your next level. Whether you're looking to upskill or reskill for the future.

## Tools Used

- Typescript
- NextJs
- Trpc
- Tailwind CSS
- Drizzle Orm
